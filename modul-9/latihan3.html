Server Pirate Bay Pindah ke 'Awan'

Siapa tak kenal Pirate Bay? Salah satu situs penyedia konten yg selalu dikunjungi para pemburu torent dan konten illegal itu, harus mencari akal agar bisa online. Salah satunya dengan memindahkan server ke 'awan'.

Lebuh Aman

Selain lebih aman dari penggrebekan pihak berwajib mengguankan cloud computing dianggap lebih mudah bagi Pirate Bay saat ingin meng-upgrade kapasitas server.
